J'ai utilisé bolt.new pour builder le frontend de Timescanner. En quelques heures, une webapp était née. Bluffant.

En remettant les mains dans le code deux mois plus tard : une immondice monumentale. L'app était belle. Ses fondations atroces. La duplication de code et le code mort étaient partout.

Ce n'est pas un problème de bolt.new. GitClear a analysé 211 millions de lignes de code entre 2020 et 2024, sur les repos de Google, Microsoft et Meta. En 2024, la fréquence des blocs dupliqués a été multipliée par 8. La réutilisation intelligente est passée de 25% des lignes modifiées en 2021 à moins de 10% en 2024.

C'est structurel. L'IA fonctionne bien sur des tâches petites, bien spécifiées, bien découpées. Sur de grands chunks de code, elle perd le fil.

On a tout réécrit. C'est à partir de là qu'on a pensé un workflow pour délivrer vite, mais de manière qualitative.
