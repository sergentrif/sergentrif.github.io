## Mode 1 — CEO signals (P1)

### IA adoption angle CEO
| Titre | Angle condensé |
|---|---|
| Mon fond veut exemples IA concrets, je n'en ai pas | Mesurer vélocité/time-to-market, pas exemples |
| 80% utilisent l'IA, rien n'a changé | Bloqué en pilote = problème d'organisation |
| Ce n'est pas un problème de technologie | IA collective casse sur défauts organisationnels |
| Offert l'IA gratuitement un an, habitudes inchangées | Adoption forcée = seule alternative à l'inertie |
| CTO a stratégie IA pour devs, pas l'équipe entière | IA individuelle ≠ feedback loop inter-métiers |
| Ce que l'équipe fait avec l'IA, vous ne savez pas | Sans TokenOps : harness privés, org n'en bénéficie pas |
| Plus vélocité IA + même supervision = plus de dette | GitClear ×8 dupliqué · DORA -7.2% stabilité |
| CTO parle agents autonomes — questions à poser | LLM probabiliste dans pipeline déterministe = cassure |
| Bonnes pratiques IA en équipe n'existent pas encore | Désangoissant — profils qui ont structuré ça n'existent pas |
